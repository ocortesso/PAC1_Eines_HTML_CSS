Parcel

Parcel és un agrupador d'aplicacions web diferenciat per la seva experiència de desenvolupador. Ofereix un rendiment ràpid i intens que utilitza processament multicorre i requereix una configuració zero.

Primer instal·leu Parcel mitjançant npm:

                
                    npm install -g parcel-bundler

Creeu un fitxer package.json al directori del projecte mitjançant:

                
                    npm init -y

El paquet pot prendre qualsevol tipus de fitxer com a punt d’entrada, però un fitxer HTML o JavaScript és un bon lloc per començar. Si enllaceu el fitxer principal de JavaScript al codi HTML mitjançant un camí d'accés relatiu, Parcel també el processarà i substituïu la referència per un URL al fitxer de sortida.

A continuació, creeu un fitxer index.html i index.js.

                
                    <html>
                    <body>
                    <script src="./index.js"></script>
                     </body>
                    </html>

NB: El paquet converteix els actius JS en ES5, que no s'executaran en el context d'una <script type="module">etiqueta, de manera que només cal que utilitzeu <script>etiquetes simples sense typeatributs al codi HTML de la vostra font.

                
                    console.log('hello world')

Parcel té un servidor de desenvolupament integrat, que reconstruirà automàticament l’aplicació a mesura que canvieu d’arxius i admet la substitució de mòduls en calent per a un desenvolupament ràpid. Apunteu-lo al fitxer d’entrada:

                
                    parcel index.html

Ara obriu http: // localhost: 1234 / al vostre navegador. Si la substitució del mòdul en calent no funciona, potser haureu de configurar l’editor . També podeu substituir el port predeterminat amb l' -p <port number>opció.

Feu servir el servidor de desenvolupament quan no tingueu el vostre propi servidor o bé l'aplicació sigui completament client. Si teniu el vostre propi servidor, podeu executar Parcel en watchmode. Això encara es reconstrueix automàticament a mesura que els fitxers canvien i admet la substitució de mòduls en calent, però no inicia un servidor web.

                
                    parcel watch index.html

També podeu utilitzar createapp.dev per crear un projecte Parcel al navegador. Seleccioneu les funcions que necessiteu, com ara Reactar, Vue, Typescript i CSS, i veureu que el projecte es genera en temps real. Podeu utilitzar aquesta eina per aprendre a configurar un nou projecte i també podeu descarregar el projecte com a fitxer ZIP i començar a codificar a l'instant.
