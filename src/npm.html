npm

npm és el registre de programari més gran del món. Els desenvolupadors de codi obert de tots els continents utilitzen npm per compartir i prestar paquets i moltes organitzacions utilitzen npm per gestionar també el desenvolupament privat.

Aquí teniu una introducció ràpida a npm:

npm consta de tres components diferents:

  • la pàgina web
  • la interfície de la línia de comandament (CLI)
  • el registre

Utilitzeu el lloc web per descobrir paquets, configurar perfils i gestionar altres aspectes de la vostra experiència npm. Per exemple, podeu configurar Orgs (organitzacions) per gestionar l'accés a paquets públics o privats.

El CLI s’executa des d’un terminal i és com la majoria dels desenvolupadors interactuen amb npm.

El registre és una gran base de dades pública de programari JavaScript i la metainformació que l'envolta.

Utilitzeu npm per. . .

  • Adapta els paquets de codi per a les teves aplicacions o incorpora paquets tal com són.
  • Descarregueu eines autònomes que podeu utilitzar immediatament.
  • Executeu paquets sense descarregar-vos amb npx .
  • Compartiu codi amb qualsevol usuari de npm, a qualsevol lloc.
  • Restringiu el codi a desenvolupadors específics.
  • Creeu Orgs (organitzacions) per coordinar el manteniment, codificació i desenvolupadors de paquets.
  • Formeu equips virtuals mitjançant Orgs.
  • Gestioneu diverses versions de dependències de codi i codi.
  • Actualitzeu les aplicacions fàcilment quan s’actualitzi el codi subjacent.
  • Descobriu diverses maneres de resoldre el mateix trencaclosques.
  • Cerqueu altres desenvolupadors que treballin en problemes i projectes similars.

Començant

Per començar amb npm, podeu crear un compte , que estarà disponible a http://www.npmjs.com/~ el vostre nom d'usuari .

Després de configurar un compte npm, el següent pas és utilitzar la interfície de línia d’ordres (CLI) per instal·lar npm . Esperem veure què crees!

Compartir paquets i col·laborar amb altres

Si trieu compartir els vostres paquets públicament, no hi ha cap cost. Per utilitzar i compartir paquets privats, heu d’actualitzar el vostre compte. Per compartir-ho amb altres, creeu organitzacions, anomenades npm Orgs , i convideu a altres a treballar amb vosaltres, de forma privada (de pagament) o públicament (de forma gratuïta). O podeu registrar-vos en una instància privada de npm per a la vostra empresa, anomenada npm Enterprise , de manera que podeu desenvolupar paquets internament que no es comparteixen públicament.

Aprèn més

Per obtenir més informació sobre npm com a producte, noves funcions i usos interessants de npm, inscriviu-vos al nostre butlletí al npm-setmanal i assegureu-vos de seguir @npmjs a Twitter.

Per a tutories, tutorials i aprenentatge, visiteu l'escola de nodes . Penseu en assistir o acollir un esdeveniment nodeschool (normalment gratuït!) En un lloc proper o bé, o bé utilitzeu les eines d’autoajuda que podeu trobar al lloc.
